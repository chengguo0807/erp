## 备注

- 所有模块开发遵循 RESTful API 规范
- 前端使用 Element Plus 组件库
- 后端使用 MyBatis-Plus 简化 CRUD
- 所有接口需要权限控制
- 重要操作需要记录操作日志
- 数据变更需要记录审计日志
